BLUENRG

BLIENRG

本学期在“无线传感技术与应用开发”课程中学习蓝牙芯片相关内容，一下是对于本课题的小结
一、准本好本课程需要的两个软件：STM32Cubemx和IAR Systems并升级到最高版本（在STM官网下载）help里找到“Manage embedded software packages”中下载固件包（stm32f4 1.24.1版本,
X-CUBE-MEMS1 6.2.0)，或在官网中下载，安装于STM32Cube–Repository目录下。
二、配置芯片蓝牙功能
1、打开STM32CUBE软件，新建工程（New project），选择芯片（STM32F401RETx），双击打开，显示芯片设置
2、蓝牙配置：在工具栏中选择Additional Software中的STMicorelectronics.X_CUBE_BLE1，并展开，在Application中选择Beacon,
HCI-TL中选择Basic,HCI-TL-INTERFACE中选择UserBoard，勾选剩余的全部选项
3、在系统配置中（System Core）配置SWD:选中SYS,Debug中选择Serial Wire。选择后，芯片相对的引脚会显示调试
4、配置全双工和串口的异步模式（Connectivity)：SPI1中的Mode中选择全双工模式，对应的工具栏中配置时钟参数，
时钟中将预分频器改为4.USART1中 Mode选择异步模式
5、将芯片PC4引脚设置为GPIO_EXTI4,系统配置中选择GPIO，勾选NVIC中的EXTI line4 interrupt
6、配置引脚：芯片中的PA4、PB1均设置为输出引脚（GPIO_Output)
7、Additional Software—Microelectrontronics X_CUBE_BLE1.4.4.0—勾选ireless BlueNRG_MS和 Wireless Application
工具栏中的设置中进行相关配置，勾选Exti line中的PC4，勾选BUS IO driver中的SPI1，勾选CS Line中的PA4，勾选Reset Linez中的PB1，勾选BSP USART中的USART1
