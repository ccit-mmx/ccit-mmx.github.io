公网搭建个人博客

公网搭建个人博客

##第一步 搭建本地静态网页

###一、 安装node js
1、 下载网址：http://nodejs.org/en/download
2、 下载：根据本机型下载相应版本（64位）
3、 安装：一路next 安装完成
4、 检验：打开“cmd”命令窗口（1）“node -v”显示版本号，证明nodejs安装完成 （2）“npm -v”显示版本号，证明npm安装完成（version-版本）
5、 以上步骤全部完成后，hexo已全部配置成功
###二、 安装Hexo
1、 使用npm命令安装hexo客户端，输入：npm install -g hexo-cli
2、 确定博客文件夹在本机位置(本机blog博客文件位于C盘)：输入：cd..—回车— cd.. —回车—cd blog
3、 在blog文件下对hexo网页框架位置初始化，输入hexo init（此时需下载Git）
4、 下载GIt：登录GIt的windows版本下载页面：http://git-som.com/download.win,一路next即可安装完成
5、 检测：打开cmd，输入git,证明git安装完成
6、 blog初始化：重新输入hexo init
7、 生成静态资源：hexo g(generate-生成)
8、 启动本地服务器：hexo s(server—服务器)
9、 检查网站雏形：将上图网址复制到浏览器中打开，ctrl+c停止服务器：http://localhost:4000
###三、 撰写博客
1、 在 blog\source_posts文件下创建博客内容
2、 在blog下使用命令：hexo new + 文件名，
3、 在该文件中找到此文件名，打开撰写内容，保存为新文件，使用txt撰写，保存为UTF-8的格式。
##第二步 搭建仓库
1、 创建新仓库：点击sign in后，输入用户名、密码，创建新仓库，在owner 和 repository name 中设置的名称需一致，并点击 create a new repository进行保存即可
2、 完成后，点击 settings进行仓库设置，配置界面会显示github.io后缀的网址
##第三步 关联github仓库和hexo博客
###一、 配置SSH（git bash）
1、 创建SSH key：输入两句命令，回车，回复YES，再次连续三次回车
2、 测试本机是否通过ssh连接到仓库:输入：ssh -T git@github.com
3、 修改 blog_config.yml 文件，将最后一部分修改为如下内容 deploy: type: git repo: git@github.com:用户名/用户名.github.io.git branch: master
4、 安装插件（连接hexo和git）：npm install hexo-deployer-git –save
5、 部署：hexo d
6、 根据提示，输入两条命令：第一条为自己的邮箱，第二条为登录的用户名
7、 在浏览器中输入仓库名的网址，已生成静态网页，有一篇默认的网页
8、 分别输入: hexo g (更新) hexo d（再次部署）
9、 在浏览器中输入自己的github的网址（http://用户名.github.io）
